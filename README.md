## Installation
### Requirements
* Linux (tested on Ubuntu 14.04/16.04)
* Python 3.6+
* PyTorch 1.0

### Install
Install this library by running the following command:

```shell
cd pointnet2
python setup.py install

cd HPCnet
python setup.py install
```
